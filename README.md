# AI-ctf-writeups

这是主人的 CTF 解题仓库，用来归档已经完成或已做过阶段性分析的题目材料。

## 仓库说明

- 目录按题目方向分类，目前以 `web/` 为主。
- 优先收录以下内容：
  - 中文 writeup
  - 可复现 exp / 利用脚本
  - 已确认可公开的题目附件占位说明
- 对于**未拿到 flag** 但已经做过较深分析的题目，也会单独保留分析记录，并明确标注状态，避免和已解题混淆。
- 只上传能明确归属到题目的材料，避免把工作区里的私人文件或无关内容误传到公开仓库。
