## 目录规则

每道题至少保留：

```text
题目目录/
├── README.md
└── writeup.md
```

其余内容**按需创建**：

- 有独立 exp 时，才创建 `exp/`
- 有题目附件时，才创建 `attachments/`
- 没有内容时，不创建空目录，也不写占位 README
